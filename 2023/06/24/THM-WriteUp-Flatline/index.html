0x01 RCE

we can use this Proof of Concept (PoC) to gain shell

Afterwards, we can use revshells.com to generate a reverse shell script.
https://www.revshells.com/

We can find two flags in C:\Users\Nekrotic\Desktop.

however, reading root.txt requires NT AUTHORITY\SYSTEM privileges.

0x02 Privilege Escalation

When we couldn’t find a solution while searching everywhere, we discovered the “projects” folder.

Within that folder, “openclinic” appears to be somewhat suspicious.
We can find the details of its vulnerabilities on https://www.exploit-db.com/exploits/50448

let’s first confirm what actions we can perform on mysqld.exe.

It seems that we can replace mysqld.exe with a fake one to obtain a reverse shell
use msfvenom to generate a reverse shell exe

After replacing mysqld.exe, we can proceed to restart the system.

nice~ we got nt authority\system now we can read root.txt hehe~
